# font-family.kr
빠르고 편리한 한글 웹 폰트 사용 방법 

## 구글 웹 폰트 사용시
### Noto Sans (고딕체, 노토)

1.

해당 CSS에 링크를 추가합니다.

@import url(//fonts.googleapis.com/earlyaccess/notosanskr.css); 

2.

해당 폰트를 아래와 같이 스타일을 지정하여 적용합니다.

font-family: 'Noto Sans KR', sans-serif; 

예시>

* {
    font-family: 'Noto Sans KR', sans-serif; 
}
